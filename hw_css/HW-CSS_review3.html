    grid และ flex แตกต่างกันอย่างไร

    Grid grid เหมาะสำหรับจัดการlayoutต่างๆในแบบ 2D
    Flex flex เหมาะสำหรับจัดการlayoutต่างๆในแบบ 1D หรือ เป็นแถวเรียงกัน

    Justify กับ Align แตกต่างกันอย่างไร

    Justify Justify ใช้ในการจัดการการเรียงcontentในแนวนอน
    Align Align ใช้ในการจัดการการเรียงcontentในแนวตั้ง

    What's Position?

    Position ใช้ระบุรูปแบบของการวาง position ของ element ต่างๆประกอบไปด้วย

    1. Static : เป็นมาตรฐานเดิมของทุก element
    2. Relative : การเลื่อนออกจากตำแหน่งเดิมเป็นจุดอ้างอิง ตามที่เรากำหนด
    3. Absolute : การเลื่อนออกจากตำแหน่งบนซ้ายสุดของหน้าเป็นจุดอ้างอิง
    4. Fixed : เป็นการยึดอยู่กับหน้าเว็บ
    5. Sticky : การรวม relative กับ fixed ไว้ด้วยกัน โดยการกำหนด relative เพิ่มว่าจะให้ไปfix
      ที่bottomหรือtop เวลาscrollเมาส์จะเห็นblockเลื่อนและถูกตรึงไว้
