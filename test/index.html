  • div+p คือ เลือก p tag แรกใน div
  • div~ul คือ เลือก ul ทั้งหมดใน div

    COMBINATOR EXAMPLE CASE

    Example Code for [div p]

    <div>
    <p>HEADING</p> แถว A
    <span><p>Lorem ipsum dolor sit amet.</p></span> แถว B
    </div>
  • ถ้าใช้ div p จะเลือกทั้งแถว A และ แถว B
  • ถ้าใช้ div > p จะเลือกแค่แถว A เพราะ B มี span ครอบอยู่
  • ถ้าใช้ div + p จะเลือกแค่แถว A เพราะ A เป็น p อันแรก
